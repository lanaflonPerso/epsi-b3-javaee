JDBC (Java DataBase Connectivity) est l'API standard pour interagir avec les bases données en Java. JDBC fait partie de l'édition standard et est donc disponible dans le JDK.

Le pilote de base de données

JDBC est une API indépendante de la base de données. Chaque fournisseur de SGBDR fournit sa propre implémentation du driver JDBC. Pour pouvoir se connecter à une base de données, il faut donc ajouter le driver (qui se présente sous la forme d'un fichier jar) dans le classpath à l'exécution du programme.
